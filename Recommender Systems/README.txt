Video Game Recommendation System Implementation:

The code for this project was written in python.

In order for this program to run, you must load in the ‘rating.txt’ file provided in the project folder.

This program will calculate the average ratings for a specified user, the cosine similarity’s of a specified user, and the recommended videos games for a specified user.

***This program uses the graphlabs package which can only be ran using Python 2***

I’ve provided screenshots the programs output for clarification of answers in the word doc.
